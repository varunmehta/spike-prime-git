# SpikePrimeGit Documentation

GitHub Pages documentation for SpikePrimeGit.

## View Documentation

Published at: `https://[your-username].github.io/spike-prime-git/`

Local: Open `index.html` or run `python -m http.server 8000`

## Publishing to GitHub Pages

1. Push `docs/` folder to repository
2. Go to Settings â†’ Pages
3. Set Source: "Deploy from a branch"
4. Select `main` branch, `/docs` folder
5. Save

Site publishes in a few minutes.

## Customization

- Edit `index.html` for content
- Edit `style.css` for styling (brand color: `#f5c402`)
- Add screenshots to `images/` folder
