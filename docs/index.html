üìñ What is SpikePrimeGit?

SpikePrimeGit is a Chrome extension that helps students and coaches automatically save LEGO SPIKE Prime projects to GitHub. This makes it easy to:

  • Version Control: Keep track of changes to your projects over time
  • Backup: Never lose your hard work - everything is safely stored in the cloud
  • Collaboration: Share projects with teammates and coaches
  • Portfolio: Build a portfolio of your robotics projects

üìã What You'll Need

  • Google Chrome browser
  • A GitHub account (free)
  • Access to LEGO SPIKE Prime software
  • About 10-15 minutes for setup

Follow these steps carefully to set up SpikePrimeGit. Don't worry - you only need to do this once!

1

Install the Chrome Extension

  1. Download the SpikePrimeGit extension files
  2. Open Chrome and go to chrome://extensions/
  3. Turn on "Developer mode" (toggle in top-right corner)
  4. Click "Load unpacked" and select the extension folder
  5. You should see the SpikePrimeGit icon appear in your Chrome toolbar

Create a GitHub Account

If you don't have a GitHub account yet:

  1. Go to github.com/signup
  2. Enter your email address and create a password
  3. Choose a username (this will be visible to others)
  4. Verify your email address
üí° Tip for Coaches: Consider creating a team organization on GitHub to manage student projects in one place.
3

Create a GitHub Repository

A repository is like a folder where your projects will be saved.

  1. Go to github.com/new
  2. Enter a repository name (e.g., "spike-prime-projects")
  3. Choose "Public" or "Private" (Private keeps your code hidden)
  4. Check "Add a README file"
  5. Click "Create repository"

4

Create a GitHub OAuth App

This allows the extension to securely connect to your GitHub account.

  1. Click the SpikePrimeGit extension icon in Chrome
  2. Copy the "OAuth Callback URL" shown in the popup
  3. Go to GitHub Developer Settings
  4. Click "OAuth Apps" ‚Üí "New OAuth App"
  5. Fill in the form:
    • Application name: SpikePrimeGit
    • Homepage URL: https://spike.legoeducation.com
    • Authorization callback URL: Paste the URL you copied earlier
  6. Click "Register application"
  7. Copy the Client ID
  8. Click "Generate a new client secret" and copy the Client Secret
‚ö†Ô∏è Important: Keep your Client Secret private! Don't share it with anyone.

Configure Your Settings

Now you're connected! Configure where your projects will be saved:

üöÄ How to Use SpikePrimeGit

Now that you're set up, here's how to use the extension every day.

Method 1: Auto-Sync (Recommended)

With Auto-Sync enabled, your projects automatically upload to GitHub when you save them in SPIKE Prime.

  1. Open SPIKE Prime software in Chrome
  2. Work on your project as normal
  3. Click "Save" in SPIKE Prime
  4. The extension automatically detects the save and uploads to GitHub
  5. You'll see a notification when the upload is complete
üí° Tip: You don't need to do anything special - just save your project like normal!

Method 2: Manual Upload

If you prefer manual control, you can upload projects yourself:

  1. In SPIKE Prime, download your project (File ‚Üí Download)
  2. The project will be saved as a .llsp file
  3. Click the SpikePrimeGit extension icon
  4. Click "Choose File" under "Upload Project"
  5. Select your .llsp file
  6. Click "Upload & Sync"

üîß Troubleshooting

The extension icon shows a red "!" badge

This means you're not connected to GitHub. Click the extension icon and click "Connect to GitHub" to reconnect.

Auto-sync isn't working

Check these things:

  • Make sure Auto-Sync is turned ON in the extension settings
  • Make sure you've selected a repository and branch
  • Try refreshing the SPIKE Prime page
  • Check that you're connected (green checkmark in the extension)

I get an error when uploading

Common solutions:

  • Check your internet connection
  • Make sure the repository and branch still exist on GitHub
  • Try disconnecting and reconnecting to GitHub
  • Check the browser console (F12) for detailed error messages

The OAuth callback URL doesn't work

Make sure you copied the exact URL from the extension. It should look like: https://[extension-id].chromiumapp.org/

I lost my Client Secret

You'll need to generate a new one:

  1. Go to your GitHub OAuth Apps
  2. Click on your SpikePrimeGit app
  3. Click "Generate a new client secret"
  4. Update the credentials in the extension

How do I start over?

To reset the extension:

  1. Click the extension icon
  2. Click "Disconnect"
  3. Click "Reconfigure OAuth App"
  4. Follow the setup steps again

Managing Student Projects

Here are some strategies for using SpikePrimeGit with your team:

Option 1: Individual Student Repositories

  • Each student creates their own GitHub account and repository
  • Pros: Students learn to manage their own code
  • Cons: Harder to monitor all students

Option 2: Team Organization (Recommended)

  • Create a GitHub Organization for your team
  • Create one repository per student or per project
  • You (coach) have access to all repositories
  • Pros: Easy to monitor, backup, and manage
  • Cons: Requires initial setup

Best Practices

  • Have students name their projects descriptively (e.g., "line-follower-v2")
  • Encourage regular saves to build up project history
  • Review commit history to see student progress over time
  • Use GitHub Issues to give feedback on projects
  • Create a shared repository for team resources and example code

Privacy Considerations

  • Students under 13 may need parental permission for GitHub accounts
  • Consider using private repositories for student work
  • Remind students not to include personal information in code comments
